## Быстрый запуск

### 1. Запуск бэкенда
```bash
cd server
npm run dev
```
Бэкенд будет доступен на `http://localhost:5000`

### 2. Запуск фронтенда (в новом терминале)
```bash
npm run dev
```
Фронтенд будет доступен на `http://localhost:5173`

## Что уже настроено:

### Фронтенд (.env создан)
- `VITE_API_URL=http://localhost:5000/api`
- `VITE_SOCKET_URL=http://localhost:5000`

### Бэкенд (.env уже существует)
- `PORT=5000`
- `MONGODB_URI=mongodb://localhost:27017/connector`
- JWT и другие настройки

## Создание админа для тестирования:
После первого запуска создайте пользователя через интерфейс, затем в MongoDB:
```javascript
use connector
db.users.updateOne(
  { email: "ваш-email@example.com" },
  { $set: { role: "admin" } }
)
```
